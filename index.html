¿Que es una Variable?

Crea un espacio en la memoria para almacenar un valor

var miVariable = "holaMundo";

console.log(miVariable);

variable sin valor: cuando una variable no tiene valor pero se muestra en consola, aparecerá el termino undefined.

var a;
console.log(a);

camelCase

Se utiliza para nombrar las variables. Es una convencion que standariza los nombres de variables con màs de una palabra. Se trabaja con la primera letra general en minuscula y luego la inicial de cada palabra en mayuscula.

Operador de Asignación

Se puede crear la variable sin asignarle valor. Al momento de asignarselo luego de creada, se debera hacer: "variable = valor"

ejemplo: a = 5;

Asignar el valor de una variable a otra

var a = 5;

var b = a;

Inicializar una variable

asignar un valor a una variable creada es inicializarla

let y const cumplen la misma funcion que var

case sensitive

los nombre de la variables son sensibles a mayusculas y minusculas, por lo que deben escribirse correctamente como se declaró y para eso usamos la convencion camelCase

Operaciones aritmeticas

suma:

var suma = 7 + 3;

resta:

var resta = 7 - 3;

Multiplicacion:
let producto = 5 * 5;
console.log(producto);

cuando se multiplica por 0, el resultado siempre sera 0

Division

let cociente = 20 / 4

var resto = 10 % 3

console.log(producto);

resultado será 1

Decimales

let decimal = 20.56;

Aumentar el valor de una variable en 1

Se puede hacer de dos formas

var librosComprados = 105;

console.log(librosComprados); el resultado sera 105

var librosComprados = librosComprados + 1;

console.log(librosComprados); el resultado sera 106

librosComprados++; solo aunmenta el valor de la variable en 1

console.log(librosComprados); el resultado sera 107

Reducir el valor de una variable en 1

Se puede hacer de dos formas

var librosComprados = 100;

console.log(librosComprados); el resultado sera 100

var librosComprados = librosComprados - 1;

console.log(librosComprados); el resultado sera 99

librosComprados--; solo reduce el valor de la variable en 1

console.log(librosComprados); el resultado sera 98

Aumentar el valor de una variable en mayor cantidad

Se sustituye la repeticion de la variable por un +=

Se puede hacer de dos formas

var n = 15;

var n = n + 5;

console.log(n); el resultado sera 20

var n =+ 5;

console.log(n); el resultado sera 20

Para realizar una resta funciona igual, solo se sustituye el operador += por -= Igualmente ocurre con la multiplicacion, que se utiliza el operador *= y la division utiliza /=

Cadenas de caracteres

Las cadenas de caracteres se definen por el uso de comillas, sean dobles "" o simples ''. Debe usarse el mismo tipo para cerrar y abrir

Escapar comillas

Si una cadena de caracteres lleva comillas dentro de ellas, JS podria detectar un error

Para escaparlas, se usa barra invertida antes de las comillas \"

Se puede evitar el error utilizando comillas simples para abrir la cadena y comillas dobles dentro de la cadena para encerrar la expresion.

var cadenaEjemplo = 'Mi cadena es "un ejemplo"'

Secuencias de escape

\' comillas simples

\" comillas simples

\\ barra inclinada

\n linea nueva

\r retorno de carro

\t tabulacion

\b retroceso

\f Salto de pagina

Concatenar cadenas

Para concatenar cadenas de caracteres, se les coloca un signo + entre cada una. Para separar cadenas con espaciado se puede colocar el espacio dentro de una de las cadenas o bien, concatenar un espacio entre ""

var nombreCompleto = "Allan" + "Turing"

var nombreCompleto = "Allan" + " " + "Turing"
