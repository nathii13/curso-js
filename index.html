¿Que es una Variable?

Crea un espacio en la memoria para almacenar un valor

console.log(miVariable);

variable sin valor: cuando una variable no tiene valor pero se muestra en consola, aparecerá el termino undefined.

var a;
console.log(a);

camelCase

Se utiliza para nombrar las variables. Es una convencion que standariza los nombres de variables con màs de una palabra. Se trabaja con la primera letra general en minuscula y luego la inicial de cada palabra en mayuscula.

Aumentar el valor de una variable en 1

Se puede hacer de dos formas

var librosComprados = 105;

console.log(librosComprados); el resultado sera 105

var librosComprados = librosComprados + 1;

console.log(librosComprados); el resultado sera 106

librosComprados++; solo aunmenta el valor de la variable en 1

console.log(librosComprados); el resultado sera 107

Reducir el valor de una variable en 1

Se puede hacer de dos formas

var librosComprados = 100;

console.log(librosComprados); el resultado sera 100

var librosComprados = librosComprados - 1;

console.log(librosComprados); el resultado sera 99

librosComprados--; solo reduce el valor de la variable en 1

console.log(librosComprados); el resultado sera 98

Aumentar el valor de una variable en mayor cantidad

Se sustituye la repeticion de la variable por un +=

Se puede hacer de dos formas

var n = 15;

var n = n + 5;

console.log(n); el resultado sera 20

var n =+ 5;

console.log(n); el resultado sera 20

Para realizar una resta funciona igual, solo se sustituye el operador += por -= Igualmente ocurre con la multiplicacion, que se utiliza el operador *= y la division utiliza /=

Cadenas de caracteres

Las cadenas de caracteres se definen por el uso de comillas, sean dobles "" o simples ''. Debe usarse el mismo tipo para cerrar y abrir

Escapar comillas

Si una cadena de caracteres lleva comillas dentro de ellas, JS podria detectar un error

Para escaparlas, se usa barra invertida antes de las comillas \"

Se puede evitar el error utilizando comillas simples para abrir la cadena y comillas dobles dentro de la cadena para encerrar la expresion.

var cadenaEjemplo = 'Mi cadena es "un ejemplo"'

Secuencias de escape

\' comillas simples

\" comillas simples

\\ barra inclinada

\n linea nueva

\r retorno de carro

\t tabulacion

\b retroceso

\f Salto de pagina

Concatenar cadenas

Para concatenar cadenas de caracteres, se les coloca un signo + entre cada una. Para separar cadenas con espaciado se puede colocar el espacio dentro de una de las cadenas o bien, concatenar un espacio entre ""

var nombreCompleto = "Allan" + "Turing"

var nombreCompleto = "Allan" + " " + "Turing"

Concatenar variables y cadenas

Es posible realizar esta acción, nada más se colocan las cadenas, se utiliza +, se llama la variable y puede colocarse en cualquier posicion.

var verbo = "programar"

var mensaje = "Estoy aprendiendo a " + verbo

console.log (mensaje)

mensaje: Estoy aprendiendo a programar

Concatenar variables

Se pueden crear cadenas concatenando variables.En este caso, se utiliza += para tomar un valor inicial, sumarle el nuevo y generar un nuevo valor.

var mensajeCompleto = "Estoy aprendiendo a "

var final = "cocinar"

mensajeCompleto += final;

console.log(mensajeCompleto)

Longitud de una cadena de caracteres

Se pueden contar los caracteres de una cadena utilizando .length dentro de console.log()

var cadenaLarga;

cadenaLarga = "Estoy en proceso de aprender";

console.log(cadenaLarga.length);

Indice de caracteres

Cada caracter cuenta con un valor. Siempre inicia con 0. Para ir a cada valor, se coloca en console.log el nombre de la variable, seguido de brackets cuadrados y dentro el valor que se busca en numeros

var lenguajeDeProgramacion = "Javascript"

console.log(lenguajeDeProgramacion[0])

El resultado será "J".

Inmutabilidad de las cadenas de caracteres

No es posible alterar un caracter dentro de la cadena. Se puede reasignar el valor completo de la cadena de caracteres. Por eso es inmutable, porque no puede cambiar

Como acceder al ultimo caracter de la cadena

Se utiliza la longitud de la cadena -1 para ir al ultimo caracter de una cadena.

var cadenaEjm2 = "programacion"

console.log(cadenaEjm2.length -1)

resultado = n

Para acceder a otros caracteres de derecha a iz, se va restando el indice. Ejm: -2 para el penultimo caracter y asi sucesivamente.

Arreglos Arrays

Los arreglos son "listas" de datos agrupadas de manera relacional. Se llaman de igual manera como una variable pero se encierran en brackets []. Cada elemento debe separarse por una coma y un espacio

Ejm: var arreglo = ["John", 24]

Arreglos anidados:

Se pueden anidar arreglos dentro de un arreglo principal. Se colocan dentro de brackets dentro de un bracket principal

Ejm: var listaDeProductos = [["Camisa", 10, "s110"], ["Celular", 989, "s23u"], ["Zapatos", 35, "t690"]]

¿Como acceder a elementos de un Array?

Se pueden llamar con el indice del elemento encerrado entre brackets []

ejemplo: var esteArreglo = [15, 29, 54,8]

console.log(esteArreglo[0])

console.log(esteArreglo[1])

console.log(esteArreglo[2])

Modificar elementos de un arreglo

Simplemente se le reasigna el valor llamnndo la variable y el indice a cambiar

ejm: var arregloDos = [10, 56, 13]

arregloDos[0] = 27

console.log(arregloDos)

Acceder a arreglos multidimensionales

Un arreglo aninado es un multidimensional. Para poder acceder a ellos se crea la variable con los arreglos anidados y con console.log se llama el indice del array y luego el indice del elemento interno. Ambos entre brackets []

var arregloMulti= [[90, 56, 78], [13, 29, 09], [25, 27, 15]];

0 1 2

console.log(arregloMulti[2][2]);

Metodos

.push

.push = Es un mètodo que permite colocar valores al final de un arreglo.

var estaciones = ["Invierno", "Otoño", "Primavera"]

estaciones.push("Verano");

console.log(estaciones)

.pop

.pop se utiliza para eliminar el ultimo elemento de un arreglo. Tambien se puede llamar a elemento eliminado asignandolo a una variable y colocandolo como variableInicial.pop()

var estaciones

estaciones = ["Invierno", "Otoño", "Primavera", "Verano"]

estaciones.pop();

console.log(estaciones)

var estaciones;

var estacion;

estaciones = ["Invierno", "Otoño", "Primavera", "Verano"]

estacion = estaciones.pop()

console.log(estaciones)

console.log(estacion)

.shift

Se utiliza para remover el primer elemento de un arreglo.

var estaciones = ["Invierno", "Otoño", "Primavera", "Verano"]

estaciones.shift();

console.log(estaciones)

.unshift

Se utiliza para colocar elementos en la primera posicion del arreglo

var estaciones = ["Invierno", "Otoño", "Primavera"]

estaciones.unshift("Verano")

console.log(estaciones)
